Operating Systems Terminology

Terms Definitions
Operating System The Software that controls the overall operation of a computer.
Job Queue A storage organization in which objects(in this case jobs) are ordered in first-in,first out fashion.
FIFO First-in and First-out, this means that the objects are removed from the queue in the order in which they arrived.
Interactive Processing A operating system that allows a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing A operating system that provides service to multiple users at the same time.
Multiprogramming Time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
Application Software Programs for performing tasks particular to the machines utilization
System Software Programs for performing tasks that are common to computer systems in general.
Utility Software System software that consists of software units collectively.
Graphical User Interface(GUI) This manipulates objects to be manipulated, such as files and programs are represented pictorially on the display as icons.
Shell An older user interface that communicated with users through textual messages using a keyboard and a monitor screen.
Kernel The internal part of an operating system.
Device Driver A component of the kernel that are software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager/td> Coordinates the use of the machine's mass storage facilities.
Memory Manager Another component of the Kernel that is charged with the task of coordinating the machines use of main memory.
Virtual Memory Large fictional memory space created by paging.
Boot Strapping A procedure that transfers the operating system from mass storage into main memory.
ROM Special non volatile memory cells, its contents can be read but not altered.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling The technique of holding data for output at a later but more convenient time.
Super User(or administrator) The person that gains highly privileged access to the operating systems by identifying him or herself as the administrator during the login process.
Multi-Factor Authentication This is when you need more than just one piece of information to gain access to an account. You may need to answer questions that only the user would know.
