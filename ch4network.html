Network and Internet Terms

Network Classifications Definitions
Lan Local area network,consists of collection of computers in a sinhlr building or buidling complex.
Man Metropolitan area network, a network of intermediate size such as one spanning a local community.
Wan Wide area network,links machines over a greater distance varying from neighboring cities to opposite sides of the world.
Network Models Definitions
Bus Network Machines are all connected to a common communication line this line is a bus.
Star Network One machine serves as a central focal point to which all the others are connected.
Protocols Definitions
CSMA/CD Ethernet This is a protocal that dictates that each message be broadcast to all the machines on the bus.
CSMA/CA The policy of trying to avoid collisions rather than stop them.
Combining Networks Definitions
Repeater A little more than a device that passes signals back and forth between the two original buses without considering the signals.
Bridge A bridge connects two buses and it looks at the destination address that accompanies each message and forwards a message across connection only when the message is destined for another computer.
Switch Switches can connect several busses rather than just two,thus a switch produces a network consisting of several buses extending from the switch as spokes on a wheel.
Router The connection between networks to form an internet is handled by routers.Routers are special purpose computers used for forwarding messages.
Gateway The point which one network is linked to an internet, serves as a pasageway between network and the outside world.
Client/Server Model Definitions
P2P:peer to peer Involves processes that provide service to and recieve service from each other.
Distributed System Software units that execute as processes on different computers.
Cloud Computing Huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing Definitions
Tier 1 Very high speed,high capacity international WANs. Known as the backbone of the Internet.
Tier 2 These are more regional in scope and less potent in their capabilities.
Tier 3 An independent internet operated by a single authority that is in the business of supplying internet access to individual homes and businesses.
IP Address Definitions
ICANN Internet Corportation for Assigned Names and Numbers, is a non-profit established to coordinate the internets operation.
DNS Directories that provide address translation services to clients.
TLD:Top Level Domain Top level domains are suffixs that are specific for certain things. For example one is .com or .gov.
Country Code TLDs This is a suffix to make it easy to tell which country the website is being published from, for example Australia is .au.
Internet Protocols Definitions
FTP File Transfer Protocol, is an application for accessing another computer from a great distance.
HTTP Hyper Text Transfer Protocol,these handle webpages.
SMTP Simple Mail Transfer Protocol, a way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web Definitions
URL Uniform Resource Locator, a unique address that each document recieves.
Client-Side Activities Activities preformed by the client(such as a browser)
Server-Side Activities Activities preformed by the server(such as a webserver)
Internet Four Layer Software Definitions
Application Layer Four layers consisting of software routines rather than poeple and business.
Transport Layer The message is sent from the application layer to this layer.
Network Layer The message is sent to this layer next where it is prepared for transmission
Link Layer The message is then recieved by this layer and passed back up until it is delivered.
Internet is:a packet switching network. Definitions
Packet Switching Network Digital data is sent by breaking into blocks of bits called packets.
Fault Tolerant The ability of a system to continue operating properly in the event of a failure of one or more componenets
Redundant More than one way to do something in this case more than one way to route data.
Cybersecurity Definitions
Malware Malicious Software
Virus A software that infects a computer by inserting itself into programs that already reside in the machine.
Worm An autonomous program that transfers itself through a network,taking up residence in a computer and sending itself to other computers.
Trojan Horse A program that enters a computer system disguised as a desirable program.
Spyware Software that collects information about activities at the computer on which it resides and reports the information back to the instagator of the attacks.
Phishing A technique of obtaining information explicitly by simply asking for it.
DDoS Attack Distributed Denial of Service,The work of generating the attacks is distributed over many hosts.
Protection Definitions
Firewall A filter for traffic passing through a point in the network.
Spam Filter Firewalls designed to block unwanted email.
Proxy Server A software unit that acts as an intermediart between a client and a server with the goal of shielding the client from adverse actions of the server.
Antivirus Software This is used to detect and remove the presence of known viruses and other infections.
Cryptography The study of sending and recieving messages securely in the presence of adversaries.
Public Key Encryption Involves the techniques by which the encryption and decryption algorithms are known and one of the keys is known, but the encoded message is secure.
