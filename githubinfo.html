All About Github

    Questions and Answers

    • What is a version control system: Software tools that help software teams manage changes to source code over time.
    • What is Git: A tool that is used for source code management.
    • What is Github: Github is a developer platform that is used to create,share,manage, and share code.
    • What is the difference between git and github: Github is a platform that uses git software, the difference between the two is that git is a tool and github is a platform for developers.
    • Who started github and how was it started: Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chacon were the founders of github. It was started as just a fun weekend project when they realized it could be much more.
    • What company owns it now: Github Inc is owned by Microsoft Corporation.
    • How much does a github account cost: 49 dollars a month.
    • What is the Octocat: The mascot of github.

    Git/Github Terms

    • Repository: A place,building,or receptacle where things are or may be stored.
    • Commit: To carry out or perpetrate
    • Fork: A new repository that shares code and visibility settings with the original “upstream” repository.
    • Push: Exert force on something to get it away from the origin of force.
    • Pull requests: A proposal to merge a set of changes from one branch into another.
    • Workflows: A system for managing repetitive processes and tasks which occur in a particular order.
    • Issues: The existence of some anomaly or flaw in a software system that requires resolution.
    • Raw button: Opens the file in a raw form, meaning that any HTML formatting disappears.
    • Blame button: Used to examine the contents of a file line by line and see when each line was last modified and who the author of the modifications was.
