Programming Languages

Terms Definitions
Machine Language A computer programming language consisting of binary or hexadecimal instructions which a computer can respond to directly.
Debugging Correcting errors in code that must be corrected.
Assembler Coverts mnemonic expressions inro machine language instructions.
Assembly Language A mnemoic system for representing programs is collectively.
Translator A program that translated programms expressed in high-level primitives into machine-languages.
Compiler Translation programs
Interpreter Similar to translators, except that they executed the intructions as they were translated instead of recording the translated version for future use.
Cross-platform Software Cross-platform software is independent of an operating system's design as well as the machines hardware design and is therefore executable throughout a network.
Natural Language Languages such as english,german, and latin.
Formal Language Programming languages.
Machine Independence A program written in a third-generation language could theoretically be used on any machine simply by applying the appropriate complier.
High-level programming language Provide more abstractions for the programmer and make it easier for people to read and write a program.
Object-oriented programming(OOP) The main aim of OOP is to bind together the data and the functions that operate on them so that no other part of the code can access this data except that function.
Scripting Language/td> Typically used to perform administrative tasks rather than to develop complex programs.
Data Types Interger:numeric data consisting of whole numbers, probably stored using two's complete notation. Float: Numeric data that might contain values other than whole numbers probably stored in float-point notation.
Primitive Data Type Such as int for integer and char for character.
Data Structure The conceptual shape or arrangement of data.
Python Collection Data Structures List:Lists are used to store multiple items in a single variable.They are created using square brackets. Tuple: A tuple is a collection which is ordered and unchangeable.Tuples are written with round brackets. Set:A set is a collection which is unordered, unchangeable*, and unindexed. Dictionary:Dictionaries are used to store data values in key:value pairs.A dictionary is a collection which is ordered*, changeable and do not allow duplicates.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local Variable A variable declared within a function.
Global Variable Variables whose scopes are not restricted to a particular part of a program.
Source Program A program in its orginal form.
Object Program An updated version of a program.
